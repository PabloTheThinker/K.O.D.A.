# K.O.D.A.

**Kinetic Operative Defense Agent** — an open-source security specialist
agent harness by Vektra Industries.

K.O.D.A. wraps scanner tooling, LLM providers, and an evidence-first audit
model into a single CLI that can do recon, incident response, audits, and
forensics without giving up operator control.

## What it is

- A security-focused agent harness. Narrow on purpose.
- Local-first: runs on your box. No managed tier, no phone-home.
- Evidence-first: every finding is chained, hashed, and reverifiable.
- Pluggable: 11 LLM providers, 8 scanner wrappers, external skill packs.
- Auditable: append-only JSONL log of every security-relevant event.

## What it is not

- Not a general-purpose coding agent.
- Not a cloud service.
- Not a replacement for your red/blue team — it's a force multiplier.
